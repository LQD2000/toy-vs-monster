# 小太阳宝宝

## 基础信息

| 属性 | 值 |
|------|-----|
| 名称 | 小太阳宝宝 |
| 类型 | 太阳能类产电 |
| 分类 | 基础单位 |
| 电力消耗 | 50 |
| 攻击力 | 0 |
| 血量 | 80 |
| 冷却时间 | 8秒 |

## 描述

可爱的太阳能小玩具，圆圆的脸蛋，头顶太阳能板。放在阳台上可以持续产生电力，是卧室场景的基础产电单位。

## 产电机制

| 属性 | 值 |
|------|-----|
| 产出间隔 | 10-15秒 |
| 产出量 | 50电力 |
| 特殊效果 | 无 |

## 获取方式

- 通关 1-1 后解锁

## 升级数值

| 等级 | 产出间隔 | 产出量 | 血量 | 升级消耗 |
|------|----------|--------|------|----------|
| 1 | 10-15秒 | 50 | 80 | - |
| 2 | 9-14秒 | 55 | 90 | 120金币 |
| 3 | 8-13秒 | 60 | 100 | 250金币 |
| 4 | 7-12秒 | 65 | 110 | 400金币 |
| 5 | 6-11秒 | 70 | 120 | 600金币 |

## 升阶

| 星级 | 条件 | 效果 |
|------|------|------|
| 2星 | Lv.5 + 木质材料x8 | 产出量+10，外观变化 |
| 3星 | Lv.10 + 木质材料x25 | 解锁分支进化 |

## 分支进化

（3星后可选择）

| 分支 | 名称 | 效果 |
|------|------|------|
| A | 阳光暴走 | 产出间隔减半，但产出量降低30% |
| B | 能量核心 | 产出量翻倍，但间隔增加50% |

## 设定

小太阳宝宝是阳台上最勤劳的小家伙，它总是笑眯眯地收集阳光，然后把温暖的电力分享给所有玩具朋友。虽然它不能战斗，但没有它，其他玩具都无法发挥实力。
