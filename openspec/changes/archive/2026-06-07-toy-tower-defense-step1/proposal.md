## Why

为玩具塔防游戏项目搭建基础开发环境，确保后续开发工作能够顺利进行。

## What Changes

**项目初始化：**
- 安装 Unity 6.3 LTS (6000.3.x) 引擎
- 创建新的 Unity 项目（2D 模板）
- 配置项目设置（分辨率、平台）
- 创建基础目录结构（Assets/Scripts、Prefabs、Scenes、Art、Audio、Resources）
- 创建实现文档目录（openspec/changes/toy-tower-defense/docs/）
- 安装必要 Package（Mirror、TextMeshPro、DOTween、New Input System）
- 初始化 Git 仓库，配置 .gitignore
- 配置 New Input System（替换旧版 Input Manager）

## Non-goals

- 不涉及游戏玩法设计或实现
- 不涉及游戏资源创建
- 不涉及游戏逻辑开发
